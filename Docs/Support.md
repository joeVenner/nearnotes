# NearNote Support

For help, bug reports, or feature requests, contact:

**ylafrimi@gmail.com**

Public support page: **https://joevenner.github.io/moments-hq/support/**

When reporting a location reminder issue, include:

- NearNote version and build number;
- iPhone model and iOS version;
- whether Location access is set to Always;
- whether Notifications are allowed;
- the steps that caused the problem.

Do not send reminder text, private addresses, passwords, or other sensitive information.
