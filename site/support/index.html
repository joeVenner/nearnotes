How can we help?

For help, bug reports, or feature requests, contact NearNote support.

Email support Report on GitHub

Include these details

  • NearNote version and build number
  • Your iPhone model and iOS version
  • Whether Location access is set to Always
  • Whether Notifications are allowed
  • The steps that caused the problem

Protect your privacy

Do not send reminder text, private addresses, passwords, or other sensitive information. NearNote’s in-app problem report lets you review diagnostic details before sharing them.

Map link problems

If a shortened Google Maps link cannot be resolved, confirm the iPhone is online and try again. You can also open the link in Google Maps, copy the full place URL, or search for the place manually in NearNote.
